# Whitespace-lang example

Reference: [Whitespace "Hello, World" example](https://web.archive.org/web/20150907210706if_/http://compsoc.dur.ac.uk/whitespace/hworld.ws)

## Warning

DO NOT ENABLE FORMATTER IN THIS WORKSPACE.

## Run

First, install dependencies:

```bash
yarn
```

Then you can run the example:

```bash
yarn start
```

or

```bash
yarn wspace ./hello.ws
```

